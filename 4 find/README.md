# Challenge

Seorang customer datang ke IBox, dia menanyakan ketersediaan produk Iphone 13 warna Sierra Blue. Sebagai seorang seller anda harus mencarinya dari data berikut:

```
const phone = [
  {
    seri: "Iphone 13",
    warna: "Grey",
    stok: 10
  },
  {
    seri: "Iphone 13",
    warna: "Red",
    stok: 20
  },
  {
    seri: "Iphone 13",
    warna: "Sierra Blue",
    stok: 0
  }
]
```

output yang diharapkan:

```
{
  seri: "Iphone 13",
  warna: "Sierra Blue",
  stok: 0
}
```
